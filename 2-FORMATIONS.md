# Badminton Doubles — Formations, Positioning & Rotation

## The Two Fundamental Formations

Every doubles rally is a constant transition between two formations. Understanding **when** and **how** to switch is what separates good pairs from great ones.

---

## 1. Attacking Formation: Front-Back

```
            NET
  ┌─────────────────────────┐
  │                         │
  │      ┌───┐              │
  │      │ A │  Front       │   A = Front player (net)
  │      └───┘  player      │
  │                         │
  │         - - - - -       │   Service line
  │                         │
  │                         │
  │      ┌───┐              │
  │      │ B │  Rear        │   B = Rear player (attack)
  │      └───┘  player      │
  │                         │
  └─────────────────────────┘
           BASELINE
```

### When to Use

Switch to front-back whenever your opponents hit the shuttle **upward** (lift, lob, clear). Your side has the smash opportunity.

### Rear Player (B) — Responsibilities

- Primary attacker: smashes, drops, half-smashes, attacking clears.
- Controls the tempo and direction of the attack.
- Covers the entire rear court area.
- Should be positioned slightly behind center, biased toward the side they're attacking from.

### Front Player (A) — Responsibilities

- Stands just behind the service line, slightly to the side where the rear player is attacking.
- Racket held **high** (net height or above), ready to intercept.
- Watches for: net kills, weak blocks, pushes through the gap.
- Does NOT turn around to watch the partner — watches the **opponents**.
- **Movement of the front player dictates the rotation.** If the front player doesn't move, the rear player can't know what's covered.

### Front Player Positioning Rule

The front player shifts laterally to follow the rear player's attack:

```
  Rear attacks RIGHT side:        Rear attacks LEFT side:

       NET                              NET
  ┌──────────────┐               ┌──────────────┐
  │              │               │              │
  │        ┌───┐ │               │ ┌───┐        │
  │        │ A │ │               │ │ A │        │
  │        └───┘ │               │ └───┘        │
  │              │               │              │
  │        ┌───┐ │               │ ┌───┐        │
  │        │ B │ │               │ │ B │        │
  │        └───┘ │               │ └───┘        │
  └──────────────┘               └──────────────┘

  A shifts RIGHT                 A shifts LEFT
  to cover straight line         to cover straight line
```

**Why?** The straight smash is the fastest and most common attack. The front player positions to cover straight returns (blocks, drives). The rear player naturally covers the cross-court angle.

## 2. Defensive Formation: Side-by-Side

```
            NET
  ┌─────────────────────────┐
  │                         │
  │                         │
  │                         │
  │   ┌───┐       ┌───┐    │
  │   │ C │       │ D │    │   C & D stand side by side
  │   └───┘       └───┘    │   ~1 step behind mid-court
  │                         │
  │                         │
  │                         │
  └─────────────────────────┘
           BASELINE
```

### When to Use

Switch to side-by-side whenever your side hits the shuttle **upward** (lift, lob, clear). The opponents now have the smash.

### Positioning Details

- Both players stand approximately **one step behind the center of the court** (mid-court area).
- Each player covers **one half** of the court width.
- **NOT at the net** — too close makes you vulnerable to smashes.
- **NOT at the baseline** — too far back makes drops unreachable.

### Defensive Stance

- Knees bent, weight on the balls of the feet.
- Racket at waist height or slightly higher.
- Loose grip — allows faster wrist reactions.
- Non-racket arm up for balance.

### Defensive Lean/Shift

Both defenders shift slightly toward the side the attack is coming from:

```
  Opponent attacks from RIGHT:    Opponent attacks from LEFT:

       NET                              NET
  ┌──────────────┐               ┌──────────────┐
  │              │               │              │
  │              │               │              │
  │              │               │              │
  │     ┌─┐ ┌─┐ │               │ ┌─┐ ┌─┐     │
  │     │C│ │D│  │               │ │C│ │D│     │
  │     └─┘ └─┘ │               │ └─┘ └─┘     │
  │              │               │              │
  └──────────────┘               └──────────────┘
  
  Both lean RIGHT                Both lean LEFT
```

**Why?** The straight smash is fastest and most common. By leaning toward it, you shorten reaction time. Cross-court smashes travel farther and give more time to reach.

## 3. Rotation: Defense → Attack

### Trigger

The defending side plays a **tight block** or **quality net shot** that forces the opponent to hit upward.

### How It Works

1. Player C plays a tight block at the net.
2. C **moves forward** to claim the front position.
3. D **moves back** behind C to cover the rear court.
4. Formation switches from side-by-side to front-back.

```
  BEFORE (defense)               AFTER (attack)

       NET                              NET
  ┌──────────────┐               ┌──────────────┐
  │              │               │              │
  │              │               │    ┌───┐     │
  │              │               │    │ C │     │  C blocked and
  │ ┌───┐ ┌───┐ │               │    └───┘     │  moved forward
  │ │ C │ │ D │ │    ──────►    │              │
  │ └───┘ └───┘ │               │    ┌───┐     │  D moved back
  │              │               │    │ D │     │
  │              │               │    └───┘     │
  └──────────────┘               └──────────────┘
```

### Key Points

- The player who makes the net shot **always** moves forward.
- The partner **reacts** by moving back — don't wait, don't hesitate.
- Think of an invisible rope connecting both players — if one moves, the other must adjust.

## 4. Rotation: Attack → Defense

### Trigger

The attacking side is forced to hit upward (lift, clear, high defensive shot). The attack is lost.

### How It Works

1. Player A (front) or B (rear) plays a lift/clear.
2. The front player **moves back** to one side of the court.
3. The rear player **moves to the other side**.
4. Formation switches from front-back to side-by-side.

```
  BEFORE (attack)                AFTER (defense)

       NET                              NET
  ┌──────────────┐               ┌──────────────┐
  │              │               │              │
  │    ┌───┐     │               │              │
  │    │ A │     │               │              │
  │    └───┘     │    ──────►    │ ┌───┐ ┌───┐ │
  │              │               │ │ A │ │ B │ │
  │    ┌───┐     │               │ └───┘ └───┘ │
  │    │ B │     │               │              │
  │    └───┘     │               │              │
  └──────────────┘               └──────────────┘
```

## 5. "Follow the Shuttle" Rule

The most important rotation principle: **follow the direction of your shot.**

| Your Shot | Your Movement | Partner's Movement |
|-----------|--------------|-------------------|
| Drop shot from rear court | Move **forward** to net | Move **back** to cover rear |
| Lift from front court | Move **back** to one side | Move to the **other side** |
| Block at net | Move **forward** / stay at net | Move **back** behind you |
| Clear from rear court | Move to **one side** (defense) | Move to **other side** |
| Drive from the side that passes the front player | Move **forward** | Move **back** |

### Critical Rule

**Never stand still after hitting.** Every shot requires movement — either toward the net, toward the rear, or into a defensive position. Standing still after hitting is the #1 amateur mistake.

---

## 6. Clockwise vs Counter-Clockwise Rotation

Rotation can flow in either direction depending on which side the shuttle comes from and which player reacts. There is no fixed direction — it depends on the rally situation.

```
  Counter-clockwise:              Clockwise:

       NET                              NET
  ┌──────────────┐               ┌──────────────┐
  │              │               │              │
  │  ←── A       │               │       A ──→  │
  │       ↓      │               │      ↓       │
  │              │               │              │
  │       B ──→  │               │  ←── B       │
  │              │               │              │
  └──────────────┘               └──────────────┘

  A moves left/forward            A moves right/forward
  B moves right/back              B moves left/back
```

### When to Choose

- Don't force a rotation direction — use the **natural flow** of the rally.
- The front player's movement initiates the rotation; the rear player adapts.
- If forcing a rotation would leave a gap, stay in position and adjust next shot.

## 7. Serving Formation

### Server's Side

```
            NET
  ┌─────────────────────────┐
  │                         │
  │    ┌───┐                │
  │    │ S │  Server        │   S = Server (close to service line)
  │    └───┘                │
  │         - - - - -       │   Service line
  │                         │
  │    ┌───┐                │
  │    │ P │  Partner       │   P = Partner (~2 racket lengths behind)
  │    └───┘                │
  │                         │
  └─────────────────────────┘
```

- Server stands close to the service line (shortest shuttle travel distance).
- Partner stands in mid-court, ready to cover the third shot.
- After serving, the server stays at the net → natural front-back formation.

### Receiver's Side

```
            NET
  ┌─────────────────────────┐
  │                         │
  │              ┌───┐      │
  │              │ R │      │   R = Receiver (close to service line,
  │              └───┘      │       diagonal from server)
  │         - - - - -       │
  │                         │
  │    ┌───┐                │
  │    │ P │                │   P = Receiver's partner (mid-court)
  │    └───┘                │
  │                         │
  └─────────────────────────┘
```

- Receiver stands close to the service line, racket up high, weight forward.
- Receiver's partner stands in mid-court on the other side, ready to adapt.

## 8. Side Drive / Flat Exchange Formation

During flat drive exchanges (neither side clearly attacking or defending), a **transitional formation** occurs:

```
            NET
  ┌─────────────────────────┐
  │                         │
  │                         │
  │    ┌───┐                │
  │    │ A │  Slightly      │   A = slightly forward
  │    └───┘  forward       │
  │                         │
  │              ┌───┐      │
  │              │ B │      │   B = slightly behind, opposite side
  │              └───┘      │
  │                         │
  └─────────────────────────┘
```

This is a **diagonal** formation — halfway between front-back and side-by-side. Both players are ready to transition to full attack or full defense depending on the next shot.

## 9. Common Formation Mistakes

| Mistake | Problem | Solution |
|---------|---------|----------|
| Both players in front-back during defense | Can't cover court width against smashes | Switch to side-by-side when you lift |
| Both players side-by-side during attack | No net pressure, can't intercept | Switch to front-back when opponents lift |
| Front player standing too central | Blocks rear player's line of attack | Shift to the side of the attack |
| Front player too close to net | Vulnerable to pushes behind them | Stand just behind service line |
| Front player too far from net | Late to intercept weak returns | Move forward, racket up |
| Both players on the same side | Half the court is empty | Imagine the rope — if one goes left, other goes right |
| Not rotating after a shot | Gaps appear in coverage | Follow the shuttle, move after every hit |
| Forcing rotation when unnecessary | Creates chaos and collisions | Use natural rally flow, don't force positions |

## 10. Quick Reference: When to Switch

| Situation | Formation |
|-----------|-----------|
| You/partner smash | Front-back (attack) |
| You/partner drop | Front-back (follow the drop forward) |
| You/partner lift/clear | Side-by-side (defense) |
| Opponent lifts/clears | Front-back (attack) |
| Opponent smashes | Side-by-side (defense) |
| Flat drive exchange | Diagonal / transitional |
| After serving (low serve) | Front-back (server at net) |
| After receiving (you lifted) | Side-by-side (defense) |
| After receiving (you pushed/drove) | Depends on the result of the third shot |
